# PWA Icons

Place your PWA icon files here with the following sizes:
- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

These icons are referenced in the manifest.json file for Progressive Web App functionality.

You can generate these from your logo.png or Icon.png file in the parent directory using an online tool like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
